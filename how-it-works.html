How it works

  1. Connect a demo profile (Connect page).
  2. Create a bet (Yes/No or Head-to-Head), set stake & expiry.
  3. Join an open bet from Home.
  4. Settle automatically (random) or via Admin.
  5. Payout distributes the pot minus 1% demo fee (treasury).
  6. Dispute if something looks wrong; Admin can resolve.
